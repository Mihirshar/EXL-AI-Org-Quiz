# Infographic Images

Place your infographic images here. The app expects the following naming convention:

```
level-{N}-a.png   → Option A infographic for Decision N
level-{N}-b.png   → Option B infographic for Decision N
```

## Required Files

| File | Description |
|------|-------------|
| `level-1-a.png` | Decision 1 - Option A |
| `level-1-b.png` | Decision 1 - Option B |
| `level-2-a.png` | Decision 2 - Option A |
| `level-2-b.png` | Decision 2 - Option B |
| `level-3-a.png` | Decision 3 - Option A |
| `level-3-b.png` | Decision 3 - Option B |
| `level-4-a.png` | Decision 4 - Option A |
| `level-4-b.png` | Decision 4 - Option B |
| `level-5-a.png` | Decision 5 - Option A |
| `level-5-b.png` | Decision 5 - Option B |

Supported formats: `.png`, `.jpg`, `.webp`

If an image is missing, a placeholder will be shown in the UI.
